# Nexus 💎  
**By QuadraDevs**
**Nexus** é um aplicativo mobile de gerenciamento de tarefas pessoais desenvolvido por quatro amigos estudantes e apaixonados. O projeto transforma a organização diária em uma jornada pelo Rift, utilizando terminologias inspiradas no jogo e uma interface baseada na estética Hextech/LoL.
Este projeto faz parte da entrega **CP5** da disciplina de **Mobile Development** na FIAP.
---
## 🎮 Skin do Projeto: Nexus Pro
- **Nome do app:** Nexus
- **Equipe de desenvolvimento:** QuadraDevs
- **Terminologia utilizada:**
  - **Status:** Pendente *(Em Lane)*, Em Andamento *(Farming)* e Concluída *(Vitória!)*.
  - **Prioridades / Elo:** Baixa *(Ferro)*, Média *(Platina)* e Alta *(Challenger)*.

## ✨ Funcionalidades
- **Autenticação de Invocador:** login simples com redirecionamento baseado no tipo de usuário, diferenciando Admin e Player.
- **Farm de Tarefas (CRUD):**
  - Cadastro de tarefas com prioridade, categoria e status.
  - Listagem de tarefas com filtros por status.
  - Edição de tarefas cadastradas.
  - Remoção de tarefas com confirmação de exclusão, no estilo “Surrender”.
- **Memória do Rift:** persistência local dos dados utilizando `AsyncStorage`.
- **Modo Rift:** suporte completo aos temas Dark e Light, com visual inspirado no universo de League of Legends.
- **Frase do Dia:** consumo de API externa para exibir frases motivacionais e incentivar o usuário durante sua jornada.
